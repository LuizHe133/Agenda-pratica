# Requisitos Funcionais

## Funcionalidades e Comportamentos do Sistema

Os requisitos funcionais descrevem o que o **sistema** deve fazer, com ênfase em componentes ou no comportamento geral. Cada requisito está identificado, classificado e possui critérios de aceitação no formato Gherkin.
